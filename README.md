# Wine Quality EDA

## Overview
This project demonstrates Exploratory Data Analysis (EDA) on a wine quality dataset using Python. The analysis includes data inspection, statistical summary, missing value detection, duplicate checking, and multiple visualization techniques to better understand the dataset.

## Features
- Dataset Analysis
- Missing and Duplicate Value Detection
- Statistical Summary
- Data Visualization using Graphs
- Correlation Analysis

## Visualizations Included
- Bar Plot
- Histogram with KDE
- Swarm Plot
- Pair Plot
- Violin Plot
- Box Plot
- Correlation Heatmap

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
```text
wine_data.xlsx
```
